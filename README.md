# Web Development Final Project

This is a professional portfolio website built as a final web development project.

## Project Structure

- `index.html` - Bootstrap-themed homepage with resume
- `expertise.html` - Graduate student expertise page on AI-assisted web development
- `scratch.html` - From-scratch page about Technology & AI Evolution
- `app.html` - Quote Collage Generator web app
- `css/` - Stylesheets (Bootstrap and custom)
- `js/` - JavaScript files (Bootstrap)
- `images/` - Image assets

## GitHub Pages Setup

### Step 1: Create GitHub Repository
1. Go to GitHub.com and create a new repository
2. Name it (e.g., "web-development-final" or "portfolio-website")
3. Make it public (required for free GitHub Pages)
4. Do NOT initialize with README (since we already have files)

### Step 2: Push Files to GitHub
Open terminal/command prompt in this project folder and run:
```bash
git add .
git commit -m "Initial commit - Web Development Final Project"
git branch -M main
git remote add origin https://github.com/[YOUR_USERNAME]/[YOUR_REPO_NAME].git
git push -u origin main
```
(Replace [YOUR_USERNAME] and [YOUR_REPO_NAME] with your actual GitHub username and repository name)

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" tab
3. Scroll down to "Pages" section (in left sidebar)
4. Under "Source", select "Deploy from a branch"
5. Select "main" branch and "/ (root)" folder
6. Click "Save"

### Step 4: Access Your Site
Your site will be available at:
- `https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/scratch.html`
- `https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/index.html`
- `https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/app.html`
- `https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/expertise.html`

It may take a few minutes for the site to become available after enabling Pages.

### Step 5: Test Everything
1. Click on every link on your site
2. Test all functionality (web app, videos, Tableau visualization)
3. Verify images load correctly
4. Test on different browsers and devices
